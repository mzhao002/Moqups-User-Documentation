# Troubleshooting

| Symptom                                           | Possible Cause                                                              | Solution                                                                                                        |
| ------------------------------------------------- | --------------------------------------------------------------------------- | --------------------------------------------------------------------------------------------------------------- |
| Cannot create a new project                       | Storage limitations on the free plan (free plan only supports two projects) | Upgrade your account to the Starter plan                                                                        |
| Need a bigger canvas because running out of space | The canvas is too small for the content being added                         | Click Format in the top-right menu, then resize the canvas by entering new width and height values          |
| Connectors are not snapping to shapes             | Connector is not properly attached to anchor points                         | Drag the connector directly onto a shape’s highlighted anchor point until it snaps into place                   |
| Exported file looks blank or incomplete           | Some elements are placed outside of the canvas area                         | Ensure all elements are fully inside the canvas before exporting                                                |
| Cannot find shape libraries or default UML tools  | Stencils panel is not opened or UML library is not enabled                  | Click Stencils in the left-side menu bar, then search for “UML Class” and enable the required shape library |
| Unable to group multiple shapes                   | Shapes are not selected before grouping                                     | Hold Shift and click to select multiple shapes, then click the Group icon in the options menu           |
